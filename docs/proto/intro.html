Structure for Management Information and Protocol Operations

The pysnmp.proto package implements data structures as defined by the Structure for Management Information RFCs (RFC 1155, RFC 1902) and SNMP protocol operations (RFC 1157, RFC 1905), and also contains a few auxiliary sub-packages targeted at simplifying operations over these SNMP protocol objects.

This package contains the following sub-packages:

  • High-level API(s) to SNMP protocol objects
  • Command-line interface to SNMP protocol objects

...and modules:

  • SNMP protocol, version 1
    • Structure of Management Information (RFC 1155)
    • Protocol Operations (RFC 1157)
    • SNMP v.1 protocol classes (joint RFC 1155 & RFC 1157)
  • SNMP protocol, version 2c
    • Structure of Management Information (RFC 1902)
    • Protocol Operations (RFC 1905)
    • SNMP v.2c protocol classes (joint RFC 1155 & RFC 1905)
  • Error recovery
    • Handling package-level errors
