Objects of the GETREQUEST, SETREQUEST, GETNEXTREQUEST, INFORMREQUEST, and GETBULKREQUEST classes

Instances of GetRequest (and similar) classes have the following additional methods:

reply(**kwargs)

Returns a Response object created in response to this request object. In particular, community names and request ID's are preserved in the response message.

The optional **kwargs may be used to pass parameters to Response object constructor.
