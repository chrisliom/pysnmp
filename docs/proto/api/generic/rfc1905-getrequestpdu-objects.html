High-level API to SNMP v.2c PDU objects

The rfc1905.GetRequestPduMixIn, rfc1905.GetNextRequestPduMixIn, rfc1905.SetRequestPduMixIn, rfc1905.InformRequestPduMixIn, rfc1905.ReportPduMixIn and rfc1905.SnmpV2TrapPduMixIn classes inherit all properties of their RFC1157 ancestors and implement no additional methods of their own.

Please, refer to rfc1157.GetRequestPduMixIn methods for more information.
