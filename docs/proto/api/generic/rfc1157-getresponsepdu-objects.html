High-level API to SNMP v.1 GetResponsePdu objects

The rfc1157.GetResponseMixIn mix-in class inherites all properties of the rfc1157.GetRequestMixIn class and implements the following high-level API methods:

apiGenGetErrorStatus()

Returns current PDU error status (as integer).

apiGenSetErrorStatus(errorStatus)

Sets errorStatus (as integer) to SNMP PDU object.

apiGenGetErrorIndex()

Returns current PDU error index (as integer).

apiGenSetErrorIndex(errorIndex)

Sets errorIndex (as integer) to SNMP PDU object.
