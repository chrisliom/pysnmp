Command-line interfaces to to SNMP protocol operations

The pysnmp.proto.cli package is a collection of sub-packages implementing various command-line oriented interfaces to SNMP protocol objects. Besides providing an out-of-the-box command-line interface to newly developed tools, these sub-packages also aim at bringing a usage-level compatibility for command-line SNMP utilities with their widely known analogs.

This package contains the following sub-packages:

  • CLI implementations
    • UCD/Net-SNMP CLI
  • Error recovery
    • Handling package-level errors
