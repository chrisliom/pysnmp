Methods of SNMP v.2c SetRequest PDU mix-in objects

The rfc1905.SetRequestMixIn mix-in class defines the following methods:

cliUcdSetArgs(args)

This method behaves the same as its v.1 analog and additionally supports the following types of command-line arguments:

  • Tag -- denotes following value type. Must be one of:
    • u -- for unsigned
    • U -- for counter64
  • Value -- value initializer
