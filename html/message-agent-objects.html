Objects of agent class

Instances of agent class have the following methods:

encode_response(encoded_oids, encoded_vals)
decode_request(message)

The encode_response method takes the lists of BER encoded Object IDs encoded_oids alone with their associated values encoded_vals and builds complete SNMP message of type 'GETRESPONSE' by applying generic.encode_bindings, generic.encode_snmp_pdu and generic.encode_message_sequence methods to the input data.

The encode_request method returns complete SNMP message as a string.

The decode_request method takes complete SNMP request (as produced by manager.encode_request method) parses out and returns a tuple of (type, encoded_oids, encoded_vals) by applying the generic.decode_message_sequence, generic.decode_snmp_pdu and generic.decode_bindings methods to the input data.

The returned values are of the same semantic and data type as used in manager.encode_request method.

Returned Object IDs and their respective values are matched against each other by their positions in the encoded_oids and encoded_vals lists.

Objects of agent class have no public instance variables.
