Objects of generic class

Instances of generic class have the following methods:

encode_bindings(encoded_oids, encoded_vals)
decode_bindings(bindings)

The encode_bindings method takes the lists of BER encoded Object IDs encoded_oids and their associated values encoded_vals, binds them altogether with BER and returns the resulted binding as a string.

The decode_bindings method takes Object IDs & values bindings (as produced by the encode_bindings method), unbinds them with BER decoder and returns a tuple of lists of BER encoded Object IDs encoded_oids and their associated values encoded_vals.

Object IDs and their respective values are matched agaist each other by their positions in the encoded_oids and encoded_vals lists.

encode_snmp_pdu(type, bindings)
decode_snmp_pdu(pdu)

The encode_snmp_pdu method takes the Object IDs and values bindings (as produced by encode_bindings method), adds some transient information with BER encoder and returns a SNMP Protocol Data Unit (PDU) of specified SNMP message type as a string.

The following type argument values are recognized:

  • 'GETREQUEST' -- SNMP 'GET' request
  • 'GETNEXTREQUEST' -- SNMP 'GETNEXT' request
  • 'GETRESPONSE' -- SNMP 'GET' response
  • 'SETREQUEST' -- SNMP 'SET' request
  • 'TRAPREQUEST' -- SNMP 'TRAP' request

The decode_snmp_pdu method takes the SNMP PDU (as produced by encode_snmp_pdu method), applies BER decoder to it, and returns a tuple of the following layout (type, request_id, error_status, error_index, bindings) whereas type and bindings are strings while the rest of this tuple components are integers.

Please, refer to SNMP protocol specification for more information on the components of this tuple.

encode_message_sequence(version, community, pdu)
decode_message_sequence(message)

The encode_message_sequence method takes the SNMP version number version (integer), SNMP community (as string) and SNMP PDU pdu (as string), produced by encode_snmp_pdu method), adds some transient information with BER encoder and returns complete SNMP message (as a string).

The decode_message_sequence method takes complete SNMP message (as produced by encode_message_sequence method), applies BER decoder to it, and returns a tuple of the following layout (version, community, pdu) where version component is of type integer while the rest are strings.

Please, refer to SNMP protocol specification for more information on the components of this tuple.

Objects of generic class have no public instance variables.
