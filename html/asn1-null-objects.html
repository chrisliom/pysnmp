Objects of the asn1.NULL class.

Instances of the asn1.NULL class have the following methods:

encode([value])

The encode method converts the content of the class instance into BER octet-stream (string). The value parameter, whenever given, must be a false value.

decode(data)

The decode method attempts to convert the data octet-stream (string) into an null value, and returns a tuple of (null_class_instance, rest).

Any unprocessed part of data octet-stream is returned in the rest string.

The data octet-stream must carry an ASN.1 NULL value or an exception would be thrown.

update(value)

The update method assigns the value to class instance payload. The value must be false.

__call__()

Class instance invocation would return currently stored payload.

__cmp__(other)

Class instances may be compared against each other. The comparation would return negative if other is greater, zero on equivalence, positive if other is less.

__repr__()

Returns the canonical string representation of the class instance.

__str__()

Returns a nice string representation of the class instance.
