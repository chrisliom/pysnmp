Objects of manager class

The instances of manager class have the following methods:

open()

Create and return connected socket object to be used for communication with remote SNMP process.

send_and_receive(message)

Attempt to send SNMP message and receive SNMP response from remote SNMP process or time out in timeout seconds. Accept and return BER encoded SNMP message (as string).

For more information on building / parsing SNMP message, please, refer to the pysnmp.message module documentation.

send(message)

Attempt to send SNMP message to remote SNMP process using socket previously initialized with open method. Accept BER encoded SNMP message (given as string).

receive()

Return BER encoded SNMP response message (as string) as received from remote SNMP process or time out in timeout seconds.

close()

Terminate SNMP session previously established with remote SNMP process.

Objects of session class have the following public instance variables:

iface

Specify the IP number of local interface SNMP engine should bind to. SNMP requests would then be originated from the interface given.

The value of the iface variable takes effect only if specified before open() method is invoked.

The iface attribute is a string representing either FQDN or IP number of desired local interface.

The default value is None what means to bind to primary local interface.

timeout

Specify for how many seconds to wait for response from SNMP agent. The timeout attribute is of floating point type.

The default is 1.0 second.

retries

Specify the number of request re-transmissions. The retries attribute is of integer type.

The default is 3 retries.
